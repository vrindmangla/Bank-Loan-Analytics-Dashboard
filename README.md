# 🏦 Bank Loan Analytics Dashboard

> A comprehensive Business Intelligence solution built using **Power BI, SQL, and DAX** to analyze loan portfolio performance, borrower behavior, credit risk, and lending trends.

---

## 📌 Project Overview

The Bank Loan Analytics Dashboard provides a complete view of loan portfolio health, lending performance, borrower characteristics, and credit risk. The project transforms raw loan data into actionable insights that help financial institutions make informed lending decisions and optimize portfolio performance.

---

## 🎯 Business Objectives

- Monitor overall loan portfolio performance.
- Analyze loan repayment and recovery trends.
- Identify high-risk borrower segments.
- Evaluate the impact of credit scores and debt-to-income ratios on defaults.
- Understand borrower demographics and lending behavior.
- Generate executive-level insights and recommendations.

### 5️⃣ Executive Insights & Recommendations

#### Key Insights
- Good loans account for the majority of the portfolio.
- Higher DTI and lower credit scores are associated with increased default rates.
- Debt Consolidation is the most common loan purpose.
- Geographic lending and default trends vary significantly across regions.

#### Recommendations
- Focus lending on borrowers with strong credit profiles.
- Strengthen underwriting standards for high-risk segments.
- Diversify portfolio exposure across states and loan purposes.
- Prioritize high-recovery loan categories.
- Implement proactive risk monitoring.
- Review credit policies regularly.
- Develop retention programs for reliable borrowers.
- Expand lending in high-performing markets.

## 📈 Key Metrics

| Metric | Description |
|----------|------------|
| Total Applications | Total number of loan applications |
| Total Funded Amount | Total loan amount disbursed |
| Total Amount Received | Total repayments collected |
| Good Loan % | Fully Paid + Current Loans |
| Bad Loan % | Charged-Off Loans |
| Recovery Rate | Amount Received ÷ Funded Amount |
| Default Rate | Charged-Off Loans ÷ Total Loans |
| Average Interest Rate | Average lending interest rate |
| Average Credit Score | Borrower creditworthiness indicator |
| Average DTI | Debt-to-Income Ratio |
